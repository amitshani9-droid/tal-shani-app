# 📱 אפליקציית ניהול התוכן של טל שני — מדריך הקמה

אפליקציית PWA (עובדת בדפדפן ובטלפון כמו אפליקציה) לניהול תוכן ברשתות החברתיות, עם מחולל פוסטים מבוסס GPT-4 וחקר שוק לשוק הישראלי.

---

## 🚀 הרצה מקומית

```bash
cd tal-shani-app
npm install
npm run dev
```

נפתח בכתובת `http://localhost:5173`.

## 🏗️ בנייה לפרודקשן

```bash
npm run build
```

הקבצים נוצרים בתיקיית `dist/` — מוכנים להעלאה.

---

## ☁️ פריסה ל-Vercel

האפשרות הכי פשוטה:
1. העלה את התיקייה ל-GitHub

4. Deploy

או דרך CLI:
```bash
npm i -g vercel

## 🔑 הגדרה ראשונית (פעם אחת)

אין צורך לגעת בקוד. אחרי שהאפליקציה עולה:

1. פותחים את האפליקציה
2. לוחצים על גלגל השיניים ⚙️ בפינה של דף הבית
3. מזינים את מפתח ה-OpenAI (מתחיל ב-`sk-...`)
   - משיגים אותו ב: https://platform.openai.com/api-keys
4. שומרים — וזהו, הכל עובד

> המפתח נשמר רק במכשיר (localStorage) ולא נשלח לשום מקום חוץ מ-OpenAI.

## 📊 חקר שוק עם נתוני Google Trends אמיתיים (אופציונלי)

ברירת המחדל: חקר השוק עובד על בסיס הידע של GPT — מספיק טוב להתחלה.

לשדרוג עם נתוני Trends חיים מישראל:

1. מקימים פרויקט Firebase (חינם)
2. מעתיקים את `server/trends-function.js` לתיקיית `functions`
3. ```bash
   cd functions
   npm install firebase-functions google-trends-api cors
   firebase deploy --only functions:trends
   ```
4. מעתיקים את כתובת ה-Function שמתקבלת
5. מדביקים אותה במסך ההגדרות של האפליקציה (שדה "שרת Google Trends")

## 💡 מה האפליקציה יודעת לעשות

- **מחולל פוסטים** — בוחרים פלטפורמה + סוג תוכן + כותבים כמה מילים, GPT כותב פוסט מלא בסגנון של טל
- **עריכה ושיפור** — אפשר לערוך ידנית או לבקש מ-GPT לשנות
- **חקר שוק** — דוח על מה חם בתחום הגיבוש בישראל, רעיונות לפוסטים, קבוצות פייסבוק לפרסום, זמני פרסום והאשטאגים
- **צרי פוסט מהרעיון** — כפתור שמעביר מרעיון בחקר השוק ישר למחולל
- **לוח תוכן** — תכנון פוסטים קדימה
- **ארכיון** — כל מה שפורסם + טיוטות שמורות

## ⚠️ הערה על פרסום בקבוצות פייסבוק

האפליקציה **לא** מפרסמת אוטומטית בקבוצות — Meta חוסמת את זה וזה מסכן את החשבון.
במקום זה היא נותנת רשימת קבוצות מומלצות + טיפים, וטל מעלה ידנית (גם נראה יותר אותנטי).

פרסום אוטומטי לדף פייסבוק, אינסטגרם ולינקדאין כן אפשרי דרך ה-API הרשמי — שלב הבא.
